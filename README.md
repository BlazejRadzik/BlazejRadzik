### 🚀 Featured Project: QuantRiskEngine

**A high-performance management platform designed for multi-asset portfolio modeling.**

**Core Features:**
* **HPC Monte Carlo Engine:** Custom C++17 module integrated via Pybind11[cite: 1]. Simulates asset price paths utilizing Geometric Brownian Motion ($dS_t = \mu S_t dt + \sigma S_t dW_t$) with OpenMP for full CPU core parallelization[cite: 1].
* **Hybrid Volatility Modeling:** Combines classic GARCH(1,1) econometrics with PyTorch LSTM neural networks to correct volatility estimations[cite: 1].
* **Risk Analytics:** Calculates Value at Risk (Parametric, Historical, Monte Carlo) and Expected Shortfall/CVaR[cite: 1].
* **Statistical Backtesting:** Automated VaR validation using Kupiec POF and Christoffersen independence tests[cite: 1].
* **Portfolio Optimization:** Mean-Variance optimization (SciPy/SLSQP) featuring dynamic bounds and EWMA covariance matrices for regime shifts[cite: 1].
* **Architecture:** FastAPI backend, SQLite caching layer (replacing yfinance bottlenecks), Streamlit frontend, and fully containerized via Docker[cite: 1].
